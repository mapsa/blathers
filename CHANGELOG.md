# Changelog

## [1.0.0a1] - 2026-04-03

### Added
- Initial alpha release
- CLI commands: `init`, `validate`, `build`, `fetch`
- 5 validators: SHACL, consistency, completeness, conventions, overlap
- Sidecar Markdown authoring with YAML frontmatter
- RDF/OWL extraction via rdflib
- Named individual extraction and display
- JSON manifest generation
- Static HTML site renderer with ReSpec-style layout
- Two-column layout with sticky sidebar TOC
- Concept hierarchy with detail panel
- Search, scroll spy, highlight on arrival
- Content negotiation config generation (Apache, nginx, w3id)
- Serialization to TTL, JSON-LD, N-Triples
- SVG diagram link injection
- Light/dark theme with system preference detection
- Stripe-inspired visual design
